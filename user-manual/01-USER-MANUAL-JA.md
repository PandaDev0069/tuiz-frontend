## はじめに

**TUIZ情報王**は、教育、研修、エンターテイメント向けに設計されたリアルタイム対話型クイズプラットフォームです。このマニュアルでは、ホスト（クイズ作成者）とプレイヤー（参加者）の両方としてプラットフォームを使用するための包括的なガイドを提供します。

### 主要な概念

- **ホスト**: クイズを作成し、ゲームセッションを管理する人
- **プレイヤー**: ルームコードを使用してゲームに参加する参加者
- **クイズ**: 質問、回答、説明を含む問題集
- **ゲーム**: クイズがプレイされているアクティブなセッション
- **ルームコード**: プレイヤーがゲームに参加するために使用する6桁の一意の識別子
- **ゲームフェーズ**: ゲームの現在の段階（待機中、カウントダウン、質問、回答など）

### プラットフォームの機能

- **リアルタイム同期**: すべてのプレイヤーが同時に同じ画面を見る
- **対話型学習**: 説明付きの多肢選択問題と○×問題
- **ライブリーダーボード**: 各質問後に更新されるリアルタイムランキング
- **ホストコントロール**: ゲームフローとタイミングの完全な制御
- **モバイル対応**: スマートフォン、タブレット、デスクトップで動作
- **インストール不要**: ウェブブラウザで直接動作

## 使い始める

### システム要件

- **モダンなウェブブラウザ**: Chrome、Firefox、Safari、またはEdge（最新版）
- **インターネット接続**: リアルタイム機能のための安定した接続
- **インストール不要**: ブラウザで直接動作

### アカウント作成

1. TUIZのホームページに移動
2. 上部ナビゲーションの**「ログイン」**をクリック
3. アカウントがない場合は**「新規登録」**をクリック
4. 登録フォームに入力：
   - **メールアドレス**: メールアドレス（必須）
   - **ユーザー名**: 3-20文字、英数字とアンダースコアのみ（必須）
   - **表示名**: 1-50文字、他のユーザーに表示される（必須）
   - **パスワード**: 最低6文字（必須）
   - **パスワード確認**: パスワードと一致する必要がある（必須）
5. オプションで**「ユーザー名を表示名として使用」**にチェック
6. **「アカウント作成」**をクリック
7. ダッシュボードに自動的にリダイレクトされます

### ログイン

1. TUIZのホームページに移動
2. **「ログイン」**をクリック
3. メールアドレスとパスワードを入力
4. **「ログイン」**をクリック
5. ダッシュボードに移動します

### アカウント復旧

パスワードを忘れた場合：

1. ログインページで**「パスワードを忘れた場合」**をクリック
2. メールアドレスを入力
3. メールでパスワードリセットの手順を確認

## ホストガイド

### ダッシュボードへのアクセス

ログイン後、自動的にダッシュボードに移動します。以下が可能です：

- **クイズライブラリの表示**: 作成したすべてのクイズを表示
- **新しいクイズの作成**: 新しいクイズの作成を開始
- **公開クイズの閲覧**: 他のユーザーが作成したクイズを発見
- **ゲームセッションの開始**: クイズからゲームを開始
- **アカウント管理**: プロフィールと設定を更新

### クイズの作成

#### ステップ1: 基本情報

1. ダッシュボードから**「新しいクイズを作成」**をクリック
2. クイズの詳細を入力：
   - **タイトル**（必須）: クイズの名前（1-200文字）
   - **説明**（必須）: クイズ内容の簡単な説明
   - **サムネイル**（オプション）: クイズを表す画像をアップロード
   - **カテゴリ**（オプション）: カテゴリを選択または入力（デフォルト：「一般」）
   - **タグ**（オプション）: 分類のための関連タグを追加（例：「科学」、「歴史」、「数学」）
   - **難易度レベル**: 次から選択：
     - 簡単
     - 中級
     - 上級
     - エキスパート
   - **公開設定**: 選択：
     - **非公開**: あなただけがこのクイズを見て使用できる
     - **公開**: 他のユーザーがこのクイズを発見して複製できる
3. **「次へ」**をクリックして進む

#### ステップ2: 問題の作成

1. **「問題を追加」**をクリックして最初の問題を作成
2. 各問題について、以下を入力：
   - **問題文**（必須）: 問題自体
   - **問題画像**（オプション）: 問題に付随する画像をアップロード
   - **問題タイプ**（必須）: 次から選択：
     - **多肢選択**: 2、3、または4つの回答オプション
     - **○×**: 正確に2つのオプション（○と×）
   - **時間設定**:
     - **問題表示時間**: 回答前に問題を表示する時間（秒）
     - **回答時間**: プレイヤーが回答する時間制限（秒）
     - **説明表示時間**: 説明を表示する時間（秒）
   - **ポイント**: 正解で獲得するポイント（デフォルト：1）
   - **難易度**: 問題の難易度レベル
   - **回答選択肢**: すべての可能な回答を入力
     - 多肢選択の場合：2-4オプション
     - ○×の場合：正確に2オプション
   - **正解**: どの回答が正しいかを選択（正確に1つ必要）
   - **説明**（オプション）:
     - **説明タイトル**: 説明の簡単なタイトル
     - **説明文**: 回答の詳細な説明
     - **説明画像**: 説明に付随する画像
3. **「保存」**をクリックして問題を追加
4. ステップ1-3を繰り返してさらに問題を追加
5. 問題ナビゲーションを使用して問題を確認および編集
6. 問題は順序インデックスを編集することで並べ替え可能
7. 問題の追加が完了したら**「次へ」**をクリック

**問題作成のヒント**:

- 明確で簡潔な問題を書く
- 画像を使用して問題をより魅力的にする
- 適切な時間制限を設定（回答には通常10-30秒）
- 常に役立つ説明を提供する
- 公開前に問題をテストする

#### ステップ3: ゲーム設定の構成

1. ゲームプレイ設定を設定：
   - **クイズコード**（オプション）: クイックアクセスのための6桁コード（100000-999999）
   - **最大プレイヤー数**: 最大プレイヤー数（1-400、デフォルト：400）
   - **問題のみ表示**: 回答選択肢を表示する前に問題を表示
   - **説明を表示**: 回答公開後に説明を表示
   - **時間ボーナス**: より速い回答にボーナスポイントを付与
   - **連続ボーナス**: 連続正解にボーナスポイントを付与
   - **正解を表示**: 公開フェーズ中に正解を表示
2. すべての設定を確認
3. **「次へ」**をクリックして進む

#### ステップ4: 確認と公開

1. クイズを確認：
   - すべての問題と回答を確認
   - 設定を確認
   - 説明が完全であることを確認
2. **「公開」**をクリックして保存し、利用可能にする
3. クイズがライブラリに保存されます

**注意**: 公開されたクイズを編集できますが、変更はアクティブなゲームには影響しません。

### ゲームの開始

#### ダッシュボードから

1. ダッシュボードに移動
2. ライブラリで使用したいクイズを見つける
3. **「ゲームを開始」**ボタンをクリック
4. **ホスト待機室**に移動します

#### クイズライブラリから

1. クイズライブラリに移動
2. クイズを選択
3. **「ゲームを開始」**をクリック
4. **ホスト待機室**に移動します

### ホスト待機室

ホスト待機室は、ゲームセッションを準備する場所です：

#### 機能

- **ルームコード表示**: 大きく読みやすい6桁コード
- **QRコード**: スキャンしてゲームを簡単に共有
- **プレイヤーリスト**: 参加したすべてのプレイヤーを表示
- **プレイヤー数**: 接続されたプレイヤーのリアルタイムカウント
- **ゲーム情報**: クイズタイトル、総問題数、設定
- **ゲーム開始ボタン**: 準備ができたらゲームを開始

#### ルームコードの共有

**共有方法**:

1. **画面表示**: プロジェクターや大画面にルームコードを表示
2. **QRコード**: プレイヤーがスマートフォンでQRコードをスキャン
3. **口頭発表**: コードを声に出して読み上げる
4. **書面**: ホワイトボードにコードを書くか、メッセージで共有

#### ゲームの開始

1. プレイヤーが参加するのを待つ（プレイヤーリストに表示されます）
2. 準備ができたら**「ゲームを開始」**をクリック
3. ゲームは3秒のカウントダウンで開始
4. ホストコントロールパネルに移動します

**ヒント**:

- 十分なプレイヤーが参加するまで待つ
- ルームコードを明確に発表
- プレイヤーに参加時間を与える（通常30-60秒）
- 遅れて参加するプレイヤーを防ぐためにルームをロックできる

### ホストコントロールパネル

ホストコントロールパネルは、ゲーム中のコマンドセンターです。ゲームフローを完全に制御できます。

#### コントロールパネルの機能

- **現在のフェーズ表示**: ゲームがどのフェーズにあるかを表示
- **問題情報**: 現在の問題番号と総問題数
- **タイマー表示**: 現在のフェーズの残り時間を表示
- **コントロールボタン**: ゲームの進行を管理
- **プレイヤー統計**: リアルタイムで回答統計を表示

#### ゲームコントロール

**問題を開始**

- **タイミング**: 各問題の開始時または説明後
- **アクション**: **「問題を開始」**をクリック
- **結果**: 問題がすべてのプレイヤーに表示される
- **タイマー**: 「問題表示時間」のカウントダウンを表示

**答えを表示**

- **タイミング**: プレイヤーが回答した後（またはタイマーが期限切れ）
- **アクション**: **「答えを表示」**をクリック
- **結果**: 正解が統計とともに表示される
- **統計**: 各回答を選択したプレイヤー数を表示

**次の問題へ**

- **タイミング**: 回答公開と説明（表示された場合）の後
- **アクション**: **「次の問題へ」**をクリック
- **結果**: 次の問題に進む
- **自動進行**: 時間制限後に自動進行するように設定可能

**ゲームを一時停止**

- **タイミング**: ゲーム中いつでも
- **アクション**: **「一時停止」**をクリック
- **結果**: ゲームが一時停止され、タイマーが停止
- **使用例**: 休憩、中断の処理

**ゲームを再開**

- **タイミング**: 一時停止後
- **アクション**: **「再開」**をクリック
- **結果**: 一時停止した場所からゲームが続行

**ゲームを終了**

- **タイミング**: ゲーム中いつでも
- **アクション**: **「ゲームを終了」**をクリック
- **結果**: ゲームが即座に終了し、最終表彰台を表示
- **使用例**: 緊急停止、技術的問題

#### プレイヤー管理

**プレイヤーを表示**

- リアルタイムで接続されているすべてのプレイヤーを表示
- プレイヤー名と接続ステータスを表示
- プレイヤー数を監視

**プレイヤーを削除**

- ゲームからプレイヤーを削除
- **タイミング**: 不正行為、技術的問題、その他の理由
- **アクション**: プレイヤー名をクリック → **「削除」**
- **注意**: 控えめに公平に使用

**ルームをロック**

- 新しいプレイヤーの参加を防ぐ
- **タイミング**: ゲームが開始された、または十分なプレイヤーがいる
- **アクション**: 設定でルームロックを切り替え
- **結果**: 新しいプレイヤーはルームコードで参加できない

### ゲーム画面（ホストビュー）

ホストとして、プレイヤーと同じ画面を見ますが、追加のコントロールがあります：

#### 1. カウントダウン画面

- **時間**: 3秒
- **表示**: 大きなカウントダウン数字（3、2、1）
- **目的**: プレイヤーを問題に備える
- **コントロール**: 問題表示フェーズに自動遷移

#### 2. 問題画面

- **表示**: 問題文と画像（提供されている場合）
- **時間**: 「問題表示時間」設定
- **目的**: プレイヤーに問題を読ませる
- **コントロール**: 回答フェーズに自動遷移

#### 3. 回答画面

- **表示**: すべての回答選択肢を含む問題
- **時間**: 「回答時間」設定
- **目的**: プレイヤーが回答を選択して送信
- **統計**: リアルタイムの回答送信数
- **コントロール**: 手動で答えを公開するか、タイマーを待つ

#### 4. 回答公開画面

- **表示**:
  - 正解がハイライト表示（通常は緑色）
  - 各回答を選択したプレイヤー数を示す統計
  - 各選択肢のプレイヤー割合
- **時間**: 設定可能（通常10秒）
- **目的**: 結果と正解を表示
- **コントロール**: 「次の問題へ」をクリックして進む

#### 5. リーダーボード画面

- **表示**:
  - 現在のランキング
  - プレイヤー名とスコア
  - ランク変更（上/下矢印）
  - 正確性パーセンテージ
- **時間**: 設定可能（通常10秒）
- **目的**: 現在の順位を表示
- **コントロール**: 「次の問題へ」をクリックして進む

#### 6. 説明画面

- **表示**:
  - 説明タイトルとテキスト
  - 説明画像（提供されている場合）
- **時間**: 「説明表示時間」設定
- **目的**: 教育的価値、なぜ答えが正しいかを説明
- **コントロール**: 「次の問題へ」をクリックして進む

#### 7. 表彰台画面（ゲーム終了）

- **表示**:
  - トップ3プレイヤー（表彰台の位置）
  - すべてのプレイヤーの最終ランキング
  - 最終スコア
  - ゲーム統計
- **時間**: 無期限（ゲームが終了）
- **目的**: 勝者を祝福し、最終結果を表示
- **コントロール**: ダッシュボードに戻るか、新しいゲームを開始

### クイズライブラリ管理

#### クイズの表示

**マイライブラリタブ**

- 作成したすべてのクイズを表示
- ステータスでフィルタ（下書き、公開、アーカイブ）
- タイトル、説明、またはタグで検索
- 日付、タイトル、またはプレイ回数で並べ替え

**公開閲覧タブ**

- 他のユーザーが作成したクイズを発見
- 難易度、カテゴリ、またはタグでフィルタ
- 特定のトピックを検索
- プレイ回数と評価を表示

#### クイズの編集

1. ライブラリでクイズを見つける
2. **「編集」**ボタンをクリック
3. 以下を変更：
   - 基本情報（タイトル、説明など）
   - 問題（追加、編集、削除、並べ替え）
   - 回答（回答選択肢を変更）
   - 設定（ゲーム設定）
4. **「保存」**をクリックして更新
5. **注意**: 変更はアクティブなゲームには影響しません

#### クイズの削除

1. ライブラリでクイズを見つける
2. **「削除」**ボタンをクリック
3. ポップアップで削除を確認
4. **警告**: このアクションは元に戻せません
5. **注意**: クイズはソフト削除され、管理者によって復元可能

#### クイズの複製

1. クイズを見つける（自分のものまたは公開閲覧から）
2. **「複製」**ボタンをクリック
3. クイズが「（コピー）」サフィックス付きでライブラリにコピーされる
4. 必要に応じて複製されたクイズを編集
5. **使用例**: 他の人のクイズをテンプレートとして使用、バリエーションを作成

#### クイズの公開

1. クイズを作成または編集
2. すべての問題に正解があることを確認
3. **「公開」**ボタンをクリック
4. クイズステータスが「公開」に変更
5. 公開されたクイズは以下が可能：
   - ゲームを開始するために使用
   - 他のユーザーが発見（公開の場合）
   - 他のユーザーが複製（公開の場合）

## プレイヤーガイド

### ゲームへの参加

#### ステップ1: ルームコードを取得

- **ホストから**: ホストが6桁のルームコードを共有
- **方法**:
  - 画面/プロジェクターに表示
  - スキャンするQRコード
  - 口頭発表
  - 書面またはメッセージ

#### ステップ2: 参加ページに移動

1. TUIZのホームページに移動
2. **「TUIZ参加」**ボタンをクリック
3. 参加ページに移動します

#### ステップ3: ルームコードと名前を入力

1. 入力フィールドに**ルームコード**（6桁）を入力
2. **プレイヤー名**を入力（これが他の人に表示される方法）
3. **「参加」**ボタンをクリック
4. **待機室**に移動します

**ヒント**:

- 認識可能な名前を使用
- ルームコードを再確認
- コードが機能しない場合、ホストに確認

### プレイヤー待機室

待機室は、ゲームの開始を待つ場所です：

#### 機能

- **ゲーム情報**: クイズタイトルと説明
- **プレイヤーリスト**: 参加した他のプレイヤーを表示
- **プレイヤー数**: プレイヤーの総数
- **ステータス**: 「ホストがゲームを開始するのを待っています」
- **ルームコード表示**: 参加したルームの確認

#### 期待されること

- ホストがゲームを開始するのを待つ
- リアルタイムで他のプレイヤーが参加するのを見る
- ホストが「ゲームを開始」をクリックするとゲームが自動的に開始
- ゲームが開始すると自動的にゲームに移動

**ヒント**:

- ページを更新しない（再参加が必要）
- ホストを辛抱強く待つ
- インターネット接続が安定していることを確認

### ゲームのプレイ

ゲームは自動的に異なるフェーズを進行します。手動でナビゲートする必要はありません - ゲームが各画面を案内します。

#### フェーズ1: カウントダウン（3秒）

- **表示内容**: 大きなカウントダウン数字（3、2、1）
- **時間**: 3秒
- **目的**: 問題に備える
- **必要なアクション**: なし、待つだけ

#### フェーズ2: 問題表示

- **表示内容**:
  - 問題文
  - 問題画像（提供されている場合）
  - 問題番号（例：「問題1/10」）
- **時間**: ホストが設定（通常10秒）
- **目的**: 問題を読んで理解する
- **必要なアクション**: 問題を注意深く読む
- **タイマー**: 残り時間を示すカウントダウン

#### フェーズ3: 回答フェーズ

- **表示内容**:
  - 問題文と画像
  - すべての回答選択肢（A、B、C、Dまたは○×）
  - クリック可能な回答ボタン
  - 残り時間を示すタイマー
- **時間**: ホストが設定（通常30秒）
- **目的**: 回答を選択して送信
- **必要なアクション**:
  1. すべての回答選択肢を読む
  2. 選択した回答をクリック
  3. 回答が自動的に送信される
  4. 時間が期限切れになる前に回答を変更できる
- **タイマー**: 残り時間を示すカウントダウン
- **送信**: 回答は次の場合に自動的に送信される：
  - 回答選択肢をクリックしたとき
  - 時間が期限切れ（最後に選択した回答が送信される）

**回答のヒント**:

- 選択する前にすべてのオプションを読む
- 時間が期限切れになる前に回答を変更できる
- 迅速に、しかし正確に回答
- 最後の瞬間まで待たない

#### フェーズ4: 回答公開

- **表示内容**:
  - 問題とすべての回答選択肢
  - **正解がハイライト表示**（通常は緑色）
  - **あなたの回答がマーク**（正解=緑のチェック、不正解=赤のX）
  - **統計**: 各回答を選択したプレイヤー数を示す棒グラフ
  - **パーセンテージ**: 各選択肢のプレイヤー割合
- **時間**: ホストが設定（通常10秒）
- **目的**: 正解と他の人の回答方法を見る
- **必要なアクション**: なし、観察するだけ
- **フィードバック**:
  - 正解だったかどうかを確認
  - 回答が他の人とどう比較されるかを見る
  - 統計から学ぶ

#### フェーズ5: リーダーボード

- **表示内容**:

  - **プレイヤー名**: すべてのプレイヤー名
  - **スコア**: 現在の合計スコア
  - **ランク変更**: 上下に移動したかどうかを示す矢印
  - **正確性**: 正解の割合
  - **総回答数**: 回答した問題数
- **時間**: ホストが設定（通常10秒）
- **目的**: 現在の順位と位置を確認
- **必要なアクション**: なし、観察するだけ
- **モチベーション**:
  - 自分の成績を確認
  - 進捗を追跡
  - 他のプレイヤーと競争

#### フェーズ6: 説明

- **表示内容**:
  - **説明タイトル**（提供されている場合）
  - **説明文**: なぜ答えが正しいかの詳細な説明
  - **説明画像**（提供されている場合）
- **時間**: ホストが設定（通常10秒）
- **目的**: なぜ答えが正しいかを学ぶ
- **必要なアクション**: なし、読んで学ぶだけ
- **教育的価値**:
  - 概念を理解する
  - 間違いから学ぶ
  - 正しい知識を強化

- 回答公開
- リーダーボード
- 説明
- （すべての問題が回答されるまで繰り返し）

#### フェーズ8: 最終表彰台（ゲーム終了）

すべての問題が回答された後：

- **表示内容**:
  - **トップ3プレイヤー**: 表彰台に表示（1位、2位、3位）

  - **最終スコア**: 獲得した合計ポイント
  - **ゲーム統計**:
    - 総問題数
    - あなたの正解数
    - あなたの正確性パーセンテージ
    - あなたの最終ランク
    - 所要時間
- **時間**: 無期限（ゲームが終了）
- **目的**: 勝者を祝福し、最終結果を表示
- **必要なアクション**: なし
- **オプション**:
  - 詳細統計を表示
  - ホームページに戻る
  - 別のゲームに参加


- **時間ボーナス**: より速い回答でボーナスポイントを獲得（有効な場合）

- **状態同期**: 再接続時にゲーム状態が同期される

### ゲームフロー制御

#### ホストコントロール

ホストはゲームの進行を完全に制御できます：

1. **問題を開始**: 各問題を開始
2. **答えを表示**: 正解と統計を表示
3. **次の問題へ**: 次の問題に進む
4. **一時停止/再開**: ゲームを一時停止または再開
5. **ゲームを終了**: ゲームを早期に終了

#### 自動遷移

一部のフェーズは自動的に遷移します：

- カウントダウン → 問題表示（3秒後）
- 問題表示 → 回答（問題表示時間後）
- 回答 → 回答公開（回答時間が期限切れ）
- 回答公開 → リーダーボード（公開時間が期限切れ）
- リーダーボード → 説明（リーダーボード時間が期限切れ）
- 説明 → 次の問題（説明時間が期限切れ）

#### 手動オーバーライド

ホストはフェーズを手動で進めることができます：

- 回答フェーズ中に「答えを表示」をクリック
- 任意のフェーズ後に「次の問題へ」をクリック
- いつでもゲームを一時停止
- いつでもゲームを終了

- **問題ごとの進行**: 制御されたペース

### 分析とインサイト

- **回答統計**: プレイヤーがどのように回答したかを確認
- **リーダーボード追跡**: プレイヤーの進捗を監視
- **ゲーム履歴**: 過去のゲームを確認
- **パフォーマンス指標**: クイズのパフォーマンスを追跡

## トラブルシューティング

### よくある問題

#### ゲームに参加できない

**問題**: ルームコードが機能しない、またはゲームが見つからない

**解決策**:

1. **ルームコードを確認**: 6桁すべてを再確認
2. **ゲームステータスを確認**: ゲームが終了していないことを確認
3. **ルームがロック**: ホストにルームがロックされているか確認
4. **ページを更新**: 参加ページを更新してみる
5. **キャッシュをクリア**: ブラウザキャッシュをクリアして再試行
6. **ホストに連絡**: ホストにコードを確認

#### 接続の問題

**問題**: ゲームがリアルタイムで更新されない、遅延、または切断

**解決策**:

1. **インターネットを確認**: 安定したインターネット接続を確認
2. **ページを更新**: ブラウザページを更新
3. **再接続**: ゲームを退出して再参加
4. **ブラウザ**: 別のブラウザを試す
5. **ネットワーク**: ファイアウォールがWebSocket接続をブロックしていないか確認
6. **モバイルデータ**: Wi-Fiからモバイルデータに切り替え（またはその逆）

#### 回答が送信されない

**問題**: 回答が記録されない、または送信されない

**解決策**:

1. **回答を選択**: 回答選択肢をクリックしたことを確認
2. **時間切れ**: 回答時間が期限切れになっていないか確認
3. **接続**: インターネット接続を確認
4. **再試行**: 回答を再度クリック
5. **更新**: 問題が続く場合はページを更新
6. **ホストに連絡**: 問題が続く場合はホストに通知

#### ゲーム画面が読み込まれない

**問題**: 空白画面、永久に読み込み中、またはエラーメッセージ

**解決策**:

1. **インターネット接続**: インターネット接続を確認
2. **ページを更新**: ハードリフレッシュ（Ctrl+F5またはCmd+Shift+R）
3. **キャッシュをクリア**: ブラウザキャッシュとCookieをクリア
4. **ブラウザを更新**: ブラウザを最新バージョンに更新
5. **JavaScript**: JavaScriptが有効になっていることを確認
6. **サポートに連絡**: 問題が続く場合はサポートに連絡

#### タイマーが機能しない

**問題**: タイマーがカウントダウンしない、または間違った時間を表示

**解決策**:

1. **更新**: ページを更新
2. **接続**: インターネット接続を確認
3. **ブラウザ**: 別のブラウザを試す
4. **同期**: タイマーは自動的に同期されるはず
5. **ホストに連絡**: ホストに通知

#### リーダーボードが更新されない

**問題**: リーダーボードが間違ったランキングを表示、または更新されない

**解決策**:

1. **待つ**: リーダーボードは各問題後に更新される
2. **更新**: ページを更新
3. **接続**: インターネット接続を確認
4. **同期**: ランキングは自動的に同期される
5. **ホストに連絡**: 問題が続く場合

### ブラウザ互換性

**推奨ブラウザ**（最新版）:

- ✅ Google Chrome
- ✅ Mozilla Firefox
- ✅ Safari（macOS/iOS）
- ✅ Microsoft Edge
- ✅ Opera

**サポートされていない**:

- ❌ Internet Explorer（すべてのバージョン）
- ❌ 古いブラウザ（2020年以前）

### モバイル使用

**サポートデバイス**:

- ✅ iOS（iPhone/iPad）- Safari
- ✅ Android - Chrome
- ✅ タブレット - すべてのモダンブラウザ

**モバイルのヒント**:

- 安定したWi-Fiまたはモバイルデータを使用
- デバイスを充電したままにする
- パフォーマンス向上のために他のアプリを閉じる
- より良い視認性のために横向きモードを使用
- ブラウザ設定でJavaScriptを有効にする

### パフォーマンスのヒント

**最良の体験のために**:

1. **安定した接続**: 信頼性の高いインターネットを使用（Wi-Fi推奨）
2. **タブを閉じる**: 不要なブラウザタブを閉じる
3. **ブラウザを更新**: ブラウザを最新に保つ
4. **キャッシュをクリア**: 定期的にブラウザキャッシュをクリア
5. **拡張機能を無効化**: 干渉する可能性のあるブラウザ拡張機能を無効化
6. **フルスクリーン**: より良い集中のためにフルスクリーンモードを使用

### ヘルプの取得

ここでカバーされていない問題が発生した場合：

1. **このマニュアルを確認**: 関連セクションを確認
2. **ブラウザコンソール**: エラーメッセージを確認（F12）
3. **ホストに連絡**: ゲームホストに支援を依頼
4. **サポート**: TUIZサポートに以下を含めて連絡：
   - 問題の説明
   - ブラウザとバージョン
   - デバイスタイプ（デスクトップ/モバイル）
   - 再現手順
   - 可能であればスクリーンショット
   - エラーメッセージ（ある場合）

## ベストプラクティス

### ホスト向け

#### ゲーム開始前

1. **クイズをテスト**: エラーをチェックするためにすべての問題を実行
2. **設定を確認**: 時間制限とゲーム設定を確認
3. **資料を準備**: 共有するルームコードを準備
4. **接続を確認**: 安定したインターネット接続を確認
5. **テスト実行**: 可能であれば友人とテストゲームを実行

#### ゲーム中

1. **明確な指示**: 開始前にプレイヤーにルールを説明
2. **プレイヤーを監視**: 接続の問題や混乱を監視
3. **時間を管理**: 適切なゲームペースを維持（速すぎず遅すぎず）
4. **プレイヤーを引き込む**: 興味深い統計についてコメント
5. **柔軟性**: 必要に応じてタイミングを調整（必要に応じて一時停止）

#### ゲーム後

1. **結果を確認**: 最終ランキングと統計を確認
2. **フィードバックを収集**: プレイヤーにフィードバックを依頼
3. **クイズを改善**: プレイヤーのパフォーマンスに基づいてクイズを更新
4. **統計を保存**: どの問題が難しかったかを記録

#### クイズ作成のヒント

1. **明確な問題**: 明確で曖昧でない問題を書く
2. **バランスの取れた難易度**: 簡単、中級、難しい問題を混ぜる
3. **良い説明**: 常に役立つ説明を提供
4. **適切なタイミング**: 現実的な時間制限を設定
5. **最初にテスト**: ゲームで使用する前にクイズをテスト

### プレイヤー向け

#### 参加前

1. **安定した接続**: 信頼性の高いインターネット接続を確認
2. **静かな環境**: 集中できる静かな場所を見つける
3. **デバイス準備**: デバイスが充電され、準備されていることを確認
4. **ルームコード準備**: ルームコードを書き留める

#### ゲーム中

1. **注意を払う**: 問題を注意深く読む
2. **迅速に回答**: 最後の瞬間まで待たない
3. **集中を保つ**: 気を散らすものを避ける
4. **説明を読む**: 説明から学ぶ
5. **楽しむ**: 学習体験を楽しむ！

#### 回答のヒント

1. **すべてのオプションを読む**: 選択する前にすべての回答選択肢を読む
2. **最初に考える**: 回答する前に問題を考える
3. **直感を信じる**: 不確実な場合は最初の直感に従う
4. **間違いから学ぶ**: 説明に注意を払う
5. **パニックにならない**: 冷静で集中を保つ

### 一般的なヒント

#### 最良の体験のために

1. **モダンブラウザを使用**: Chrome、Firefox、またはSafariの最新版
2. **安定したインターネット**: モバイルデータよりもWi-Fi推奨
3. **フルスクリーン**: より良い集中のためにフルスクリーンモードを使用
4. **気を散らすものを閉じる**: 他のアプリとタブを閉じる
5. **辛抱強く**: 自動遷移を待つ

#### 技術的

1. **JavaScriptを有効化**: プラットフォームが動作するために必要
2. **WebSocketを許可**: WebSocket接続が許可されていることを確認
3. **ブラウザを更新**: ブラウザを最新に保つ
4. **キャッシュをクリア**: 問題が発生した場合にキャッシュをクリア
5. **ファイアウォールを確認**: ファイアウォールが接続をブロックしていないことを確認
