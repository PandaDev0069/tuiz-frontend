## Introduction

**TUIZ情報王** is a real-time interactive quiz platform designed for education, training, and entertainment. This manual will guide you through using the platform as both a host (quiz creator) and a player (participant).

### Key Concepts

- **Host**: The person who creates quizzes and manages game sessions
- **Player**: Participants who join games using room codes
- **Quiz**: A collection of questions with answers and explanations
- **Game**: An active session where a quiz is being played
- **Room Code**: A unique identifier that players use to join a game

## Getting Started

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- No installation required - works directly in your browser

### Creating an Account

1. Navigate to the TUIZ homepage
2. Click **"ログイン"** (Login)
3. Click **"新規登録"** (Register) if you don't have an account
4. Fill in:
   - Email address
   - Username (3-20 characters, alphanumeric and underscores only)
   - Display name (1-50 characters)
   - Password (minimum 6 characters)
   - Confirm password
5. Optionally check "次回からログイン情報を記憶する" (Remember me)
6. Click **"アカウント作成"** (Create Account)

### Logging In

1. Go to the homepage
2. Click **"ログイン"** (Login)
3. Enter your email and password
4. Click **"ログイン"** (Login)

## Host Guide

### Accessing the Dashboard

After logging in, you'll automatically be taken to your dashboard where you can:

- View your quiz library
- Create new quizzes
- Manage existing quizzes
- Start game sessions

### Creating a Quiz

#### Step 1: Basic Information

1. Click **"新しいクイズを作成"** (Create New Quiz)
2. Fill in quiz details:
   - **Title**: Name of your quiz
   - **Description**: Brief description of the quiz
   - **Thumbnail**: Upload an image (optional)
   - **Tags**: Add relevant tags for categorization
   - **Difficulty**: Select difficulty level
   - **Visibility**: Choose public or private

#### Step 2: Create Questions

1. Click **"問題を追加"** (Add Question)
2. Enter question details:
   - **Question Text**: The question itself
   - **Question Image**: Upload an image (optional)
   - **Question Type**:
     - Multiple Choice (2, 3, or 4 options)
     - True/False
   - **Time Settings**:
     - Show Question Time: How long to display the question
     - Answering Time: Time limit for answering
   - **Answer Choices**: Enter all possible answers
   - **Correct Answer**: Select the correct answer
   - **Explanation**: Add explanation text (optional)
3. Click **"保存"** (Save) to add the question
4. Repeat to add more questions
5. Use navigation to review and edit questions

#### Step 3: Configure Settings

1. Set game settings:
   - **Player Limit**: Maximum number of players
   - **Question Order**: Sequential or random
   - **Show Leaderboard**: After each question or at the end
   - **Auto-advance**: Automatically move to next question
2. Review all settings

#### Step 4: Publish

1. Review your quiz
2. Click **"公開"** (Publish) to save and make it available

### Starting a Game

1. From your dashboard, select a quiz from your library
2. Click **"ゲームを開始"** (Start Game)
3. You'll be taken to the **Host Waiting Room**
4. Share the **Room Code** with participants
5. Wait for players to join
6. Click **"ゲームを開始"** (Start Game) when ready

### Managing a Game Session

#### Host Control Panel

The control panel provides real-time game management:

- **Current Question Display**: Shows the current question to all players
- **Answer Screen**: Displays when players are answering
- **Answer Reveal**: Shows correct answer and statistics
- **Explanation Screen**: Displays question explanation
- **Leaderboard**: Shows player rankings
- **Podium**: Final rankings at game end

#### Game Controls

- **Start Question**: Begin the current question
- **Reveal Answer**: Show the correct answer
- **Next Question**: Advance to the next question
- **Pause/Resume**: Pause or resume the game
- **End Game**: End the game session early

#### Player Management

- **View Players**: See all connected players
- **Player Count**: Real-time player count
- **Kick Player**: Remove a player from the game (if needed)
- **Lock Room**: Prevent new players from joining

### Quiz Library Management

#### Viewing Your Quizzes

- **My Library**: View all your created quizzes
- **Public Browse**: Browse quizzes created by others
- **Search**: Search quizzes by title, tags, or description
- **Filter**: Filter by difficulty, tags, or date

#### Editing Quizzes

1. Find the quiz in your library
2. Click **"編集"** (Edit)
3. Make changes to questions, settings, or basic info
4. Click **"保存"** (Save) to update

#### Deleting Quizzes

1. Find the quiz in your library
2. Click **"削除"** (Delete)
3. Confirm deletion

#### Cloning Quizzes

1. Find a quiz (yours or public)
2. Click **"複製"** (Clone)
3. The quiz will be copied to your library
4. Edit as needed

## Player Guide

### Joining a Game

1. Go to the TUIZ homepage
2. Click **"TUIZ参加"** (Join TUIZ)
3. Enter the **Room Code** provided by the host
4. Enter your **Name** to join
5. Click **"参加"** (Join)

### Playing the Game

#### Waiting Room

- Wait for the host to start the game
- See other players joining
- View game information

#### Question Screen

1. **Question Display**: Read the question carefully
2. **Answer Selection**: Click on your answer choice
3. **Time Limit**: Answer before time runs out
4. **Submit**: Your answer is automatically submitted when time expires or you click submit

#### Answer Reveal

- See the correct answer
- View your answer and whether it was correct
- See statistics showing how many players chose each answer

#### Explanation Screen

- Read the explanation for the question
- Understand why the answer is correct
- Wait for the next question

#### Leaderboard

- View current rankings
- See your position
- Track your progress

#### Podium (Game End)

- See final rankings
- View your final score
- Celebrate winners!

### Game Flow

The game automatically progresses through these stages:

1. **Waiting Room** → Host starts game
2. **Question Display** → Question shown to all players
3. **Answering Phase** → Players select and submit answers
4. **Answer Reveal** → Correct answer and statistics shown
5. **Explanation** → Explanation displayed (if available)
6. **Leaderboard** → Rankings updated
7. **Next Question** → Repeat from step 2
8. **Podium** → Final results at game end

## Troubleshooting

### Common Issues

#### Cannot Join Game

- **Problem**: Room code not working
- **Solution**:
  - Verify the room code is correct
  - Check if the game has started
  - Ensure the room is not locked
  - Try refreshing the page

#### Connection Issues

- **Problem**: Game not updating in real-time
- **Solution**:
  - Check your internet connection
  - Refresh the page
  - Clear browser cache
  - Try a different browser

#### Answer Not Submitting

- **Problem**: Answer not being recorded
- **Solution**:
  - Ensure you selected an answer
  - Check if time has expired
  - Try selecting again
  - Refresh if persistent

#### Game Screen Not Loading

- **Problem**: Blank screen or loading forever
- **Solution**:
  - Check internet connection
  - Refresh the page
  - Clear browser cache
  - Contact support if issue persists

### Browser Compatibility

**Recommended Browsers:**

- Google Chrome (latest version)
- Mozilla Firefox (latest version)
- Safari (latest version)
- Microsoft Edge (latest version)

**Not Supported:**

- Internet Explorer (any version)

### Mobile Usage

TUIZ works on mobile devices:

- iOS Safari
- Android Chrome
- Mobile-responsive design
- Touch-friendly interface

### Getting Help

If you encounter issues not covered here:

1. Check your internet connection
2. Try refreshing the page
3. Clear browser cache
4. Try a different browser
5. Contact support with:
   - Description of the issue
   - Browser and version
   - Steps to reproduce
   - Screenshots if possible

## Best Practices

### For Hosts

1. **Prepare Your Quiz**: Review questions before starting
2. **Test First**: Run a test game with a friend
3. **Clear Instructions**: Explain rules to players
4. **Monitor Players**: Watch for connection issues
5. **Manage Time**: Keep game pace appropriate

### For Players

1. **Stable Connection**: Use reliable internet
2. **Pay Attention**: Read questions carefully
3. **Answer Promptly**: Don't wait until last second
4. **Have Fun**: Enjoy the learning experience!
